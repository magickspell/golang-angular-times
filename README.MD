# Time Table App

This is fullstack app that can store and operate with time table.
<br>
<br>
<b>Langs</b>: Typescript, Golang.
<br>
<b>Techies</b>: Angular, MongoDB, Docker.
<br>
<b>Reqs</b>: Docker.
<br>
<hr>

### How to start local:
```
docker-compose up -d
```

#### App listening ports by default:
1) fontend:  4200.
2) backend:  8080.
3) mongodb: 8081/27017.
